# ContractEase Demo Script (3-5 Minutes)

## 1. Introduction

"This is ContractEase, a SaaS-style contract workflow system. It allows users to create contracts, capture signatures, and download finalized PDF documents."

## 2. Create Contract

- Log in as a user.
- Open the contract creation page.
- Enter contract details:
  - Title
  - Client name/email
  - Amount and currency (₹, $, €)
  - Due date
  - Description and clause options

Talk track: "At this stage, I can quickly define all core contract details in one flow."

## 3. Save Draft

- Click Save Draft.
- Show that the contract appears in the Draft section on the user dashboard.

Talk track: "Draft saving helps the user pause and continue later without losing progress."

## 4. Edit Draft

- Open the draft from dashboard.
- Update one or two fields (for example amount or due date).
- Save changes.

Talk track: "Drafts remain editable until the contract is sent."

## 5. Creator Signs

- In the create/edit flow, add the creator signature.

Talk track: "ContractEase requires creator signature before sending to the client."

## 6. Send to Client

- Send the contract.
- Show status moving out of draft into the sent/pending workflow.

Talk track: "Once sent, the client can review and complete their side of signing."

## 7. Client Signs

- Log in as client.
- Open the pending contract.
- Add client signature and submit.

Talk track: "Client signature finalizes the agreement process."

## 8. Download PDF

- Return to dashboard.
- Download the signed contract PDF.

Talk track: "The system generates a final PDF using WeasyPrint and stores it locally for download."

## 9. Show Dashboard Status Changes

- Show contract movement across dashboard states:
  - Draft
  - Pending
  - Signed
  - Declined (if applicable)

Talk track: "The dashboard gives a clear live view of every contract stage from creation to completion."
